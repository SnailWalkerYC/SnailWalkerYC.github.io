vim的使用极简版：


```
跳页 ctrl w-hjkl: left, lower, upper, right
```


```
分页 ctrl w-vs: horizontal split, vertical split.
```


```
Tab切换：
gt: next tab
gT: prior tab
1gt: first tab
```
